# hexcrawl

A browser-based Underdark hexcrawl map generator.

## Run

Open `/tmp/workspace/jonassode/hexcrawl/index.html` in a browser.

The app keeps a deterministic `seed` in the URL query string, generates the
same map for the same seed, and provides a right-click menu for:

- Show/Hide symbols
- Export as PNG
- Generate new map
